# ⚡ ANTI-KUDDUS PROTOCOL (L-Alpha)

> **Status:** Operational | **Target:** Class Captain Kodu Kuddus | **Security Level:** Maximum Anonymity

Welcome to the **Anti-Kuddus Protocol**, a highly interactive, gamified tactical resistance platform built using **React** and **Tailwind CSS**. Designed specifically for the oppressed students of Section B (Class 7) to survive, document, and neutralize the authoritarian regime of Class Captain "Kodu Kuddus".

---

## 📝 Project Description

The **Anti-Kuddus Protocol** is an immersive Single Page Application (SPA) that acts as an underground digital command center. In a classroom ruled by an aggressive captain who extorts tiffins, enforces illegal "washroom entry taxes", and blackmails students, the resistance needed a centralized hub to fight back. 

Instead of a boring, traditional software layout, this application is styled entirely as a dark cyberpunk tactical interface. It allows students to map classroom vulnerabilities, track stolen assets, verify rumors, and signal for emergency help without external routing overhead. Everything runs seamlessly under one global state container, allowing instantaneous switching between operations.

## ⚡ Core Operations & Deep Technical Breakdown

### 1. 🎛️ Underground Access Terminal (`/login` & `/signup`)
*   **The Concept:** Traditional registration forms risk exposing student identities to the captain's spy network. This portal treats authentication as a resistance enlistment pipeline.
*   **Mechanisms:** 
    *   **Identity Obfuscation:** Replaces real names with unique, non-identifiable "Secret Codenames" (e.g., *Tiffin_Avenger_07*).
    *   **Tactical Sectoring:** Operatives select their physical location in class—*Front Benchers (The Meatshields)*, *Middle Benchers (The Watchers)*, or *Back Benchers (The Core Resistance Engine)*.
    *   **Fast Bypass PIN:** Features a 4-digit rapid PIN input for standard students to log in quickly before the teacher turns around, alongside a separate high-clearance command portal for commanders Biltu and Miltu.

### 2. 🔲 Tactical Seating Planner & Line-of-Sight Laser (`/seating-planner`)
*   **The Concept:** A dynamic grid map engineered to control classroom spatial awareness and optimize defensive positioning.
*   **Mechanisms:**
    *   **Dynamic Dimension Engine:** Generates an interactive $N \times M$ grid layout representing physical classroom desks based on active user configurations.
    *   **Interactive LoS Tracking:** Implements an automated SVG vector line mapping the exact path from the *Teacher's Podium* to any hovered desk.
    *   **Visibility Coefficient:** Dynamically calculates how exposed a seat is to the teacher's sightline, adjusting risk percentages in real-time to help operatives find dead zones.
    *   **Target Locking:** Visually isolates the "Kuddus Target" cell with a pulsing high-contrast neon red border (`animation-pulse`) and custom alert tags.

### 3. 🚨 Quantum SOS Flare Portal & Command Center (`/sos` / `/captain-ops`)
*   **The Concept:** The network's primary emergency broadcast infrastructure, linking students in immediate distress with leadership intervention.
*   **Mechanisms:**
    *   **System-Wide Siren Strobe:** Activating the central interactive Panic Button triggers a global state change, flashing the entire interface in a deep crimson emergency matrix grid.
    *   **Audio Visualizer Badge:** Displays a mock animated waveform labeled "SIREN BEACON ACTIVE".
    *   **Contextual Threat Meter:** A slider control (Levels 1–5) allowing students to rate threat gravity (from *Kuddus is roaming near the bench* to *Tiffin under active siege*).
    *   **Tactical Action Pop-up:** Provides commanders Biltu and Miltu with quick-action responses, such as deploying distraction teams using fake homework scripts.

### 4. 🧮 NCTB Camouflage Override Switch (`/emergency`)
*   **The Concept:** The ultimate security failsafe designed to completely mask the application if a authority figure suddenly approaches the workstation.
*   **Mechanisms:**
    *   **Instant Theme Swap:** One click or a hotkey (`ESC`) immediately dumps the dark cyberpunk CSS configuration and forces a bright, clean, highly authentic NCTB (National Curriculum and Textbook Board) digital classroom environment.
    *   **Academic Content Spoof:** Displays realistic mathematical assets, including equations (e.g., $2x + 5 = 15$), coordinate plane references, and definition blocks for Class 7 Linear Algebra.
    *   **Decoy Interactions:** Includes a fake input box labeled "Enter Solve Value for $x$" so anyone glancing at the monitor assumes the student is intensely studying.
    *   **Secret Backdoor:** Bypassing the camouflage requires executing a concealed keystroke combination or double-tapping a specific hidden vector point.

### 5. 📉 Extortion Analytics & Black-Market Ledger (`/ledger` & `/analytics`)
*   **The Concept:** A forensic financial ledger documenting the regime's corruption and converting stolen resources into resistance economics.
*   **Mechanisms:**
    *   **Double-Entry Corruption Logger:** Tracks two distinct metrics—the **2-Taka Washroom Entry Tax** (monetary extortion) and **Stolen Calories** (tiffin theft).
    *   **Jhalmuri Currency Converter:** A custom algorithmic utility that translates extorted monetary amounts into local black-market commodities (e.g., *100 BDT Extorted = 10 Plates of Jhalmuri / 5 Cricket Bat Swings*).
    *   **Visual Forensics:** Rendered using clean, high-contrast bar charts and pie graphs to show trends in extortion spikes over the school week.
